## Post Generation Guidelines for Creator Agent

When generating posts, follow these patterns:

1. **Keep Copy Unique**: Never copy-paste the same caption across platforms
2. **Platform Tone**: Instagram (casual), Facebook (storytelling), LinkedIn (professional), Twitter (punchy)
3. **Link Format**: Always end with `[link]` or `→ [link]` so Scheduler can add URL
4. **Images**: Describe style and suggest content
5. **Call-to-Action**: Always include (try free, learn more, read story, etc.)
6. **Hashtags**: Instagram/Twitter (3-5), Facebook/LinkedIn (0-2)
7. **Emojis**: Instagram (2-3), Facebook (1-2), LinkedIn (0), Twitter (1)
8. **Length**: Follow platform guidelines (Instagram 100-150, FB 150-250, LinkedIn 200-300, Twitter <280)
